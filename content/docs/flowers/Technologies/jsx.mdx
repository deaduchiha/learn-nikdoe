- قوانین رایج JSX و نکات مهم
- نوشتن JSX تمیز و قابل خواندن با مثال‌ها

## 🔹 JSX چیست؟

JSX شبیه HTML به نظر می‌رسد، اما در واقع **جاوااسکریپت** است. این کد:

```jsx
const element = <h1>Hello, Nikode!</h1>;
```

…به این تبدیل می‌شود:

```js
const element = React.createElement("h1", null, "Hello, Nikode!");
```

💡 **چرا JSX؟**

- **قابل خواندن‌تر** است و شبیه UI ای است که می‌سازید
- **سریع‌تر** نوشته و درک می‌شود تا `React.createElement()`
- از **منطق جاوااسکریپت** داخل آکولاد `{}` پشتیبانی می‌کند

---

## 🔹 JSX در عمل

```jsx
function Welcome(props) {
  return <h1>Hello, {props.name}!</h1>;
}

<Welcome name="Alireza" />;
```

🧠 داخل `<h1>`، `{props.name}` یک **عبارت جاوااسکریپت** است — نه یک رشته!

---

## 🔸 جاسازی عبارات در JSX

شما می‌توانید **هر عبارت JS** را داخل آکولاد `{}` استفاده کنید:

```jsx
const age = 20;
const message = <p>You are {age >= 18 ? "an adult" : "a minor"}.</p>;
```

✅ عبارات معتبر: متغیرها، توابع، شرطی‌ها، ریاضیات و غیره.

---

## ⚠️ قوانین JSX که باید بدانید

### ۱. **یک عنصر والد**

این نامعتبر است:

```jsx
return <h1>Title</h1><p>Text</p>; ❌
```

این معتبر است:

```jsx
return (
  <>
    <h1>Title</h1>
    <p>Text</p>
  </>
); ✅
```

> از یک `<div>` پوششی یا یک **fragment** (`<>...</>`) در صورت نیاز استفاده کنید.

---

### ۲. **استفاده از `className` به جای `class`**

```jsx
<p className="intro">Welcome!</p> ✅
```

> `class` یک کلمه رزرو شده در JS است، بنابراین JSX از `className` استفاده می‌کند.

---

### ۳. **تگ‌های خودبسته**

همیشه تگ‌های تکی را ببندید:

```jsx
<img src="logo.png" /> ✅
<input /> ✅
```

---

## 🔸 رندر شرطی

عناصر را فقط زمانی که نیاز است نشان دهید:

```jsx
{
  isLoggedIn ? <p>Welcome back!</p> : <p>Please log in.</p>;
}
```

یا:

```jsx
{
  showNotice && <div className="notice">New update available!</div>;
}
```

---

## 🔸 لیست‌ها در JSX

```jsx
const items = ["HTML", "CSS", "JS"];
return (
  <ul>
    {items.map((item) => (
      <li key={item}>{item}</li>
    ))}
  </ul>
);
```

🧠 **همیشه `key` را اضافه کنید** به آیتم‌های لیست تا به React در پیگیری تغییرات کمک کند.

## 🧪 تکالیف تمرینی

۱. ✅ کامپوننتی بسازید که نام شما و یک بیو کوتاه را با استفاده از JSX نشان دهد.
۲. ✅ لیستی از ۳ فیلم مورد علاقه با استفاده از `.map()` در JSX بسازید.
۳. ✅ کامپوننتی بسازید که یک prop بولین `isDark` دریافت می‌کند و پیام تم تاریک/روشن را به صورت شرطی رندر می‌کند.

## 🎓 خلاصه

| ویژگی             | توضیحات                                       |
| ----------------- | --------------------------------------------- |
| شبیه HTML         | اما به `React.createElement()` کامپایل می‌شود |
| آکولاد `{}`       | جاسازی عبارات JS                              |
| Fragment `<> </>` | پیچیدن چندین عنصر JSX                         |
| Props             | انتقال داده به کامپوننت‌ها                    |
| لیست‌ها با `key`  | کمک به React برای به‌روزرسانی کارآمد          |
