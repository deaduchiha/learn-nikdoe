### ðŸ”¹ Modules: `import` and `export`

**math.js**

```js
export const add = (a, b) => a + b;
export const subtract = (a, b) => a - b;
```

**main.js**

```js
import { add, subtract } from "./math.js";

console.log(add(10, 5)); // 15
console.log(subtract(10, 5)); // 5
```

> Modules = cleaner and reusable code across files.

### ðŸ”¹ Classes

```js

  constructor(name) {
    this.name = name;
  }

  greet() {
    return `Hi, I'm ${this.name}`;
  }

const user = new Person("Nikode");
console.log(user.greet());
```

> Think of classes like blueprints for objects with logic.
