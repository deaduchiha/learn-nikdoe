## ğŸ”¹ What Is React DOM?

## ğŸ”¸ React DOM API (React 18+)

React 18 introduced **concurrent rendering** and replaced `ReactDOM.render()` with `ReactDOM.createRoot()`.

```js
// Old (React 17)
ReactDOM.render(<App />, document.getElementById("root"));

// New (React 18+)
const root = ReactDOM.createRoot(document.getElementById("root"));
root.render(<App />);
```

> âœ… Always use `createRoot` for full React 18+ features like **suspense**, **transitions**, and **streaming**.
