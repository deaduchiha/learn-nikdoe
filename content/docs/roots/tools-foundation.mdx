## Visual Studio Code (VS Code)

Visual Studio Code (VS Code) یک ویرایشگر رایگان، قدرتمند و آسان‌استفاده است که به شما کمک می‌کند کد را به طور کارآمد بنویسید.

### مراحل نصب:

- به [code.visualstudio.com](https://code.visualstudio.com/) بروید.
- نصب‌کننده مناسب سیستم عامل خود را دانلود کنید (Windows، macOS، Linux).
- نصب‌کننده را اجرا کنید و مراحل ساده را دنبال کنید.
- پس از نصب، VS Code را باز کنید و رابط کاربری دوستانه آن را کاوش کنید.

### افزونه‌های ضروری:

برای بهبود بهره‌وری، این افزونه‌های توصیه شده را نصب کنید:

- **Prettier**: کد را به طور خودکار فرمت می‌کند.
- **Live Server**: به شما امکان پیش‌نمایش سریع صفحات HTML در مرورگر را می‌دهد.
- **ESLint**: به پیدا کردن و رفع خطاهای کد JavaScript کمک می‌کند.

## Node.js LTS

Node.js برای اجرای JavaScript خارج از مرورگر، نصب کتابخانه‌ها و مدیریت پکیج‌ها ضروری است.

### مراحل نصب:

- به [nodejs.org](https://nodejs.org/en) بروید.
- نسخه LTS (پشتیبانی بلندمدت) را برای پایداری انتخاب کنید.
- نصب‌کننده دانلود شده را اجرا کنید و دستورالعمل‌های روی صفحه را دنبال کنید.
- نصب را با باز کردن terminal و تایپ کردن تأیید کنید:

  ```bash
  node -v
  npm -v
  ```

  این دستورات نسخه‌های نصب شده Node.js و npm را نمایش می‌دهند و نصب موفق را تأیید می‌کنند.

### بهترین شیوه:

همیشه از نسخه LTS برای پایداری production استفاده کنید و از نسخه‌های تجربی پرهیز کنید.

## Git و GitHub

Git برای ردیابی تغییرات در کد، همکاری و مدیریت حرفه‌ای پروژه‌ها ضروری است.

### نصب Git:

- به [git-scm.com](https://git-scm.com/downloads) بروید و Git را برای سیستم عامل خود دانلود کنید.
- Git را با تنظیمات پیش‌فرض نصب کنید (توصیه شده).
- نصب خود را با تایپ کردن `git --version` در terminal بررسی کنید.

### دستورات پایه Git:

- راه‌اندازی repository جدید:

```bash
git init
```

- اضافه کردن فایل‌ها به ردیابی:

```bash
git add .
```

- ثبت تغییرات:

```bash
git commit -m "Your commit message"
```

- اتصال به GitHub:

### راه‌اندازی GitHub:

- یک حساب رایگان در [github.com](https://github.com) ایجاد کنید.
- روی "New Repository" کلیک کنید، نام آن را بگذارید و دستورات ارائه شده را برای اتصال Git محلی به GitHub دنبال کنید.
