Visual Studio Code (VS Code) is a free, powerful, and easy-to-use editor that helps you write code efficiently.

### Installation Steps:

- Visit [code.visualstudio.com](https://code.visualstudio.com/).
- Download the installer suitable for your operating system (Windows, macOS, Linux).
- Run the installer and follow the simple prompts to install VS Code.
- Once installed, open VS Code and explore its friendly interface.

### Essential Extensions:

To improve productivity, install these recommended extensions:

- **Prettier**: Formats code automatically.
- **Live Server**: Allows you to quickly preview HTML pages in your browser.
- **ESLint**: Helps find and fix errors in JavaScript code.

## üöÄ 2. Node.js LTS

Node.js is essential for running JavaScript outside of your browser, installing libraries, and managing packages.

### Installation Steps:

- Go to [nodejs.org](https://nodejs.org/en).
- Select the LTS (Long-Term Support) version for stability.
- Run the downloaded installer and follow the on-screen instructions.
- Verify the installation by opening your terminal and typing:

  ```bash
  node -v
  npm -v
  ```

  These commands display the installed Node.js and npm versions, confirming successful installation.

### Best Practice:

Always use the LTS version for production stability and avoid experimental versions.

## üß© 3. Git & GitHub

Git is essential for tracking changes in your code, collaborating, and managing projects professionally.

### Installing Git:

- Visit [git-scm.com](https://git-scm.com/downloads) and download Git for your OS.
- Install Git with default settings (recommended).
- Check your installation by typing `git --version` in your terminal.

### Basic Git Commands:

- Initialize a new repository:

  ```bash
  git init
  ```

- Add files to tracking:

  ```bash
  git add .
  ```

- Commit changes:

  ```bash
  git commit -m "your commit message"
  ```

- Connect to GitHub:

  ```bash
  git remote add origin your_repo_link
  git push -u origin main
  ```

### Setting up GitHub:

- Create a free account at [github.com](https://github.com).
- Click ‚ÄúNew Repository,‚Äù name it, and follow the provided commands to connect your local Git to GitHub.

## üåç 4. Browser Developer Tools

Developer tools are built-in tools in browsers like Chrome and Firefox for debugging, modifying, and understanding your web pages.

### How to Open:

- **Chrome/Firefox:** Press `F12` or `Cmd + Option + J` (macOS).

### Essential Tabs:

- **Elements:** Inspect and edit HTML/CSS directly in your browser.
- **Console:** Write JavaScript commands and see logs/errors.
- **Network:** Monitor and debug network requests.

### Best Practices:

- Always test your UI changes here before changing your code.
- Regularly check the console for errors and warnings.
