## Visual Studio Code (VS Code)

**VS Code** یک ویرایشگر رایگان، سبک و در عین حال بسیار قدرتمند است که تقریبا همه توسعه‌دهندگان فرانت‌اند از آن استفاده می‌کنند. با VS Code می‌توانید کدنویسی سریع‌تر، منظم‌تر و راحت‌تری داشته باشید.

### مراحل نصب

1. وارد سایت [code.visualstudio.com](https://code.visualstudio.com/) شوید.
2. نسخه مناسب سیستم‌عامل خود را دانلود کنید (Windows، macOS یا Linux).
3. نصب‌کننده را اجرا کرده و مراحل ساده نصب را دنبال کنید.
4. بعد از نصب، VS Code را باز کنید و رابط کاربری آن را بررسی کنید.

### افزونه‌های پیشنهادی

برای اینکه تجربه بهتری داشته باشید، این افزونه‌ها را نصب کنید:

- **Prettier** → فرمت خودکار کد برای مرتب نگه داشتن فایل‌ها.
- **Live Server** → اجرای سریع HTML در مرورگر و مشاهده تغییرات در لحظه.
- **ESLint** → کمک به پیدا کردن و رفع خطاهای جاوااسکریپت.

## Node.js (نسخه LTS)

**Node.js** به شما اجازه می‌دهد جاوااسکریپت را خارج از مرورگر اجرا کنید. همچنین برای نصب کتابخانه‌ها و مدیریت پکیج‌ها با **npm** ضروری است.

### مراحل نصب

1. وارد سایت [nodejs.org](https://nodejs.org/en) شوید.
2. نسخه **LTS (پشتیبانی بلندمدت)** را انتخاب کنید (پایدارتر و مناسب‌تر از نسخه‌های آزمایشی است).
3. نصب‌کننده را دانلود و اجرا کنید.
4. بعد از نصب، ترمینال را باز کرده و تایپ کنید:

```bash
node -v
npm -v
```

اگر نسخه‌ها نمایش داده شدند یعنی نصب موفق بوده است. ✅

### نکته مهم

برای پروژه‌های واقعی همیشه از نسخه **LTS** استفاده کنید و از نسخه‌های تجربی پرهیز کنید.

## Git و GitHub

**Git** ابزاری است برای مدیریت تغییرات کد (Version Control) و **GitHub** پلتفرمی است برای ذخیره و همکاری روی پروژه‌ها. این دو در کنار هم ستون فقرات کار تیمی در برنامه‌نویسی محسوب می‌شوند.

### نصب Git

1. وارد سایت [git-scm.com](https://git-scm.com/downloads) شوید.
2. Git را برای سیستم‌عامل خود دانلود و با تنظیمات پیش‌فرض نصب کنید.
3. نصب را با دستور زیر در ترمینال بررسی کنید:

```bash
git --version
```

### دستورات پایه Git

- ایجاد یک ریپازیتوری جدید:

```bash
git init
```

- اضافه کردن فایل‌ها به استیج:

```bash
git add .
```

- ثبت تغییرات:

```bash
git commit -m "First Commit"
```

### راه‌اندازی GitHub

1. یک حساب رایگان در [github.com](https://github.com) بسازید.
2. روی **New Repository** کلیک کنید و یک ریپازیتوری جدید ایجاد کنید.
3. دستوراتی که GitHub به شما می‌دهد را در ترمینال اجرا کنید تا ریپازیتوری محلی به GitHub متصل شود.
