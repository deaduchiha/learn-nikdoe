# مسیر یادگیری مبتدی تا پیشرفته با Nikode

سلام! این راهنما از سال‌ها تجربه کمک به مبتدیان کامل برای تبدیل شدن به حرفه‌ای‌های فرانت‌اند ساخته شده است. همه چیز درباره درک واقعی مفاهیم و ساخت چیزهای جالب در طول مسیر است. با سرعت خودتان پیش بروید—وقتی چیزها برایتان روشن می‌شود جلو بروید، نه فقط چون یک برنامه زمانی می‌گوید.

> 💡 **نحوه پیشرفت:** یک مفهوم را در هر زمان انتخاب کنید، چیز کوچکی با آن بسازید، سپس تکرار کنید. سرعت شما فقط متعلق به خودتان است.

## یادگیری HTML و مباحث مربوطه به Accessibility

- **مفاهیم:** نقاط عطف معنایی، فرم‌ها، سلسله مراتب عناوین، `alt`، نقش‌های ARIA، مدیریت فوکوس، ترتیب خواندن منطقی.
- **پروژه کوچک:** یک صفحه **"درباره من"** با `<header>`، `<nav>`، `<main>`، `<footer>`؛ عبور از **WCAG AA** کنتراست رنگ.
- **راهنماها:** [MDN HTML Elements](https://developer.mozilla.org/en-US/docs/Web/HTML/Element)، [Web.dev Accessibility](https://web.dev/learn/accessibility/).

## یادگیری CSS

- **مفاهیم:** cascade و inheritance، box-model، selectors و specificity، Flexbox، Grid، **CSS Nesting & Layers** (۲۰۲۵)، logical properties، media و container queries، BEM / utility naming.
- **پروژه کوچک:** بازسازی یک کارت محصول از سایت تجارت الکترونیک pixel-perfect؛ fluid کردن تا ۳۲۰ پیکسل.
- **راهنماها:** [Kevin Powell Flexbox + Grid](https://www.youtube.com/@KevinPowell)، [Modern CSS—Stephanie Eckles](https://moderncss.dev/).

## اصول JavaScript

- **مفاهیم:** syntax ES2024 (modules، optional chaining، nullish coalescing)، انواع داده، scope و closures، متدهای arrays / objects، destructuring و spread، template literals، DOM، events، fetch API، async/await، JSON.
- **پروژه‌های کوچک:**

  ۱. **سنگ-کاغذ-قیچی** در vanilla JS + DOM.
  ۲. **داشبورد آب و هوا** — fetch کردن OpenWeather API، نمایش پیش‌بینی ۵ روزه با حالت‌های loading و error.

- **راهنماها:** [JavaScript.info](https://javascript.info/)، [_Eloquent JavaScript_ (آنلاین)](https://eloquentjavascript.net).

> ✅ **انضباط:** زود و مکرر commit کنید، کد self-documenting بنویسید، **ESLint + Prettier** را روی save فعال کنید.

## درک ساختار Design و مدیریت Layout

- **موضوعات:** Custom properties، `calc()` / `clamp()`، fluid type و spacing scales، dark-mode tokens، responsive design با **Tailwind CSS v4** و **CSS functions**.
- **پروژه کوچک:** بازسازی داشبورد آب و هوا به Tailwind؛ پیاده‌سازی system dark mode (`prefers-color-scheme`) و ساخت یک کتابخانه component کوچک با **shadcn/ui**.
- **راهنماها:** [Tailwind Docs](https://tailwindcss.com/docs)، [Smashing Mag Fluid Type](https://www.smashingmagazine.com/2022/01/modern-fluid-typography-css-clamp/).

## شروع مسیر React

- **مفاهیم:** Vite، JSX، props vs state، hooks (`useState`، `useEffect`، `useMemo`)، component composition، keys و lists، conditional render، React DevTools.

- **راهنماها:** [react.dev Tutorial](https://react.dev/learn)، [Fireship React ۱۹ "در ۱۰۰ ثانیه"](https://youtu.be/).
